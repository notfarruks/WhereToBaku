## WhereToBaku

An Expo + React Native app (with Expo Router) for discovering interesting places in Baku.  
This repo is currently a scaffold with the main directories, TypeScript setup, and minimal UI/screens wired up.

### Scripts

- **Start development server (Expo)**:

```bash
npm install
npm run start
```

Then follow the Expo CLI instructions to open the app on a device or emulator.

### Project Structure

- **app/**: Expo Router entry and screens (navigation-based).
- **components/**: Reusable UI components (cards, headers, badges, etc.).
- **features/**: Feature-specific logic, services, and types (places, filters, user).
- **hooks/**: Custom React hooks for location, places, filters, favorites.
- **lib/**: External services and configuration (Supabase, constants, mock data).
- **styles/**: Theming primitives like colors, spacing, typography.
- **assets/**: Images, icons, fonts.
